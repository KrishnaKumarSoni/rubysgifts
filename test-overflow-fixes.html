Responsive Test Instructions

Resize your browser window to test:
  • Desktop (>1024px): Cards should display with larger fonts and spacing
  • Tablet (768-1024px): Cards should adapt with medium sizing
  • Mobile (480-768px): Cards should stack and use compact spacing
  • Small Mobile (<480px): Cards should use minimal spacing and smaller fonts
Check for:
  • ✅ Text overflow handled with ellipsis
  • ✅ Images maintain aspect ratio
  • ✅ Cards don't break layout
  • ✅ Consistent spacing across elements
  • ✅ Touch-friendly button sizes on mobile
  • ✅ Broken images show placeholder icon
